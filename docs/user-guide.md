## 1. การเข้าสู่ระบบ

### วิธีเข้าสู่ระบบ
1. เปิดเว็บ HAIT Tracker
2. ที่แถบด้านซ้าย (Sidebar) จะมีปุ่ม **"Sign in with Google"**
3. คลิกปุ่มแล้วเลือกบัญชี **@up.ac.th** ของท่าน
4. ระบบจะแสดงชื่อ, รูป, และ badge สิทธิ์ของท่าน

### สิทธิ์ผู้ใช้
| สิทธิ์ | สัญลักษณ์ | สิ่งที่ทำได้ |
|-------|---------|-----------|
| ผู้ใช้ทั่วไป | 👤 ผู้ใช้ | ดูข้อมูล, แก้สถานะ, แก้ผู้รับผิดชอบ |
| Admin | 🛡️ Admin | + แก้ชื่อรายการ, วันครบกำหนด, เพิ่ม/ลบรายการ |
| Super Admin | 👑 Super Admin | + จัดการสิทธิ์ Admin |

### หมายเหตุ
- อนุญาตเฉพาะอีเมล **@up.ac.th** เท่านั้น
- Session จะหมดอายุหลัง 24 ชั่วโมง ต้อง login ใหม่
- ไม่ต้อง login ก็ดูข้อมูลได้ แต่แก้ไขไม่ได้

## 2. หน้า Dashboard

### ส่วนประกอบ
- **แถบแจ้งเตือน**: แสดงจำนวนรายการเลยกำหนดและใกล้ครบกำหนด
- **ความคืบหน้ารวม**: แสดง % รวมของทั้ง 7 หมวด
- **การ์ดสถานะ**: 4 การ์ด (ยังไม่เริ่ม, กำลังดำเนินการ, เสร็จแล้ว, ต้องปรับปรุง) — คลิกเพื่อดูรายการ
- **การ์ดหมวด**: 7 การ์ดแสดง progress แต่ละหมวด — คลิกเพื่อเข้าดู

### ปุ่มส่งออก
- **📄 PDF** — ดาวน์โหลดรายงานสรุป PDF
- **📊 Excel** — ดาวน์โหลดรายงาน Excel (แยก sheet ตามหมวด)
- **📁 คลังเอกสาร** — เปิดโฟลเดอร์ Google Drive กลาง

## 3. การใช้งาน CategoryView

### การเปลี่ยนสถานะ
1. เข้าหมวดที่ต้องการ (คลิกจาก sidebar หรือ Dashboard)
2. ในคอลัมน์ **"สถานะ"** จะมี dropdown ให้เลือก:
   - ยังไม่เริ่ม (เทา)
   - กำลังดำเนินการ (เหลือง)
   - เสร็จแล้ว (เขียว)
   - ต้องปรับปรุง (แดง)
3. เลือกสถานะใหม่ → ระบบบันทึกอัตโนมัติ

### การเปลี่ยนผู้รับผิดชอบ
- ในคอลัมน์ **"ผู้รับผิดชอบ"** เลือกจาก dropdown

### สีของแถว
- **ขอบซ้ายแดง** = เลยกำหนดแล้ว
- **พื้นหลังเขียวอ่อน** = เสร็จแล้ว
- **พื้นหลังเทาเมื่อ hover** = กำลังชี้อยู่

### การเปิดเอกสาร
- **📂 เปิด** → เปิดโฟลเดอร์ Google Drive ของหมวดนั้น
- **📎 ดู** → เปิดเอกสารตัวอย่างอ้างอิง

## 4. Gantt Chart

### การใช้งาน
- แสดง timeline ตั้งแต่ 1 เม.ย. ถึง 31 พ.ค. 2569
- **เส้นแดง** = วันนี้
- **แถบสี** = ระยะเวลาของแต่ละรายการ (สีตามสถานะ)
- เลื่อนเมาส์ไปที่แถบเพื่อดูรายละเอียด

### การซูม
- ใช้ปุ่ม **+** / **-** ที่มุมบนขวาเพื่อปรับขนาด
- ปุ่ม **📄 PDF** เพื่อ export เป็นไฟล์ PDF

## 6. สำหรับ Admin

### สิ่งที่ Admin ทำได้เพิ่มเติม
- **แก้ชื่อรายการ**: คลิกที่ชื่อ → พิมพ์ชื่อใหม่ → Enter
- **แก้วันครบกำหนด**: คลิกที่วันที่ → ใส่เลขวัน (1-30 = เม.ย., 31-61 = พ.ค.)
- **แก้ช่วงเวลา**: คลิกที่ช่วง → แก้วันเริ่มต้น-สิ้นสุด
- **เพิ่มรายการ**: คลิก "เพิ่มรายการ" ที่ด้านล่างตาราง
- **ลบรายการ**: คลิก 🗑️ ที่คอลัมน์ "ลบ" (ต้องยืนยัน)

## 7. สำหรับ Super Admin

### การจัดการสิทธิ์
1. คลิก **"⚙️ จัดการสิทธิ์ Admin"** ที่ sidebar
2. จะเห็นตารางรายชื่อ admin ทั้งหมด

### การเพิ่ม Admin
1. กรอกอีเมล @up.ac.th
2. กรอกชื่อ (ไม่บังคับ)
3. เลือก role: Admin หรือ Super Admin
4. คลิก "เพิ่มสิทธิ์"

### การเปลี่ยน Role
- ใช้ dropdown ในคอลัมน์ "Role" เพื่อสลับระหว่าง Admin และ Super Admin

### การลบ Admin
- คลิก 🗑️ แล้วยืนยัน
- **ข้อจำกัด**: ไม่สามารถลบ Super Admin คนสุดท้ายได้

## 8. การ Export PDF/Excel

### PDF
- **Dashboard**: รายงานสรุปรวมทั้ง 7 หมวด + รายละเอียดทุก item + หน้าลายเซ็น
- **CategoryView**: รายงานเฉพาะหมวดที่เลือก
- **Gantt Chart**: ภาพ Gantt Chart เป็น PDF
- ทุก PDF มี: หัวกระดาษ, ชื่อผู้ออกรายงาน, เลขหน้า

### Excel
- Sheet 1: รายการทั้งหมด
- Sheet 2-8: แยกตามหมวด HAIT 1-7
- Sheet 9: สรุปสถานะ

### Keyboard Shortcuts
| ปุ่มลัด | การทำงาน |
|---------|---------|
| Ctrl+P | ดาวน์โหลด PDF สรุป |
| Esc | ปิด modal / ยกเลิกแก้ไข |

## 9. FAQ

### Q: ทำไมเปลี่ยนสถานะแล้วไม่บันทึก?
**A:** ตรวจสอบว่าได้ login แล้วหรือยัง ถ้ายังไม่ login จะแก้ไขข้อมูลไม่ได้

### Q: ทำไม login ไม่ได้?
**A:** ต้องใช้อีเมล @up.ac.th เท่านั้น ถ้ายังไม่ได้สิทธิ์ ติดต่อ Super Admin

### Q: Admin Mode เดิมหายไป?
**A:** ระบบเปลี่ยนมาใช้ Google OAuth แทน ไม่ต้องใส่ password แล้ว

### Q: ข้อมูลหายหรือเปล่า?
**A:** ข้อมูลบันทึกอยู่ใน Google Sheets — มีประวัติการเปลี่ยนแปลงทั้งหมด

### Q: Export PDF แสดงภาษาไทยผิดปกติ?
**A:** PDF ใช้ฟอนต์มาตรฐาน ภาษาไทยบางตัวอาจแสดงผลไม่สมบูรณ์ แนะนำใช้ Excel แทนสำหรับรายงานที่ต้องการภาษาไทยครบ

### Q: ติดต่อผู้พัฒนา?
**A:** ติดต่อ Super Admin (phatcharapon.ud@up.ac.th)
