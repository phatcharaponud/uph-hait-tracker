## คำอธิบายเต็ม / Full description (สูงสุด 4000 ตัวอักษร)

```
KneeCare Companion — เพื่อนคู่ใจผู้ป่วยข้อเข่าเสื่อมและผู้ป่วยผ่าตัดข้อเข่าเทียม

โดยทีมแพทย์และนักกายภาพบำบัด โรงพยาบาลมหาวิทยาลัยพะเยา

🦵 สำหรับใคร
• ผู้ป่วยข้อเข่าเสื่อม (Knee Osteoarthritis)
• ผู้ป่วยที่กำลังจะผ่าตัดเปลี่ยนข้อเข่าเทียม (Total Knee Arthroplasty)
• ผู้ที่ผ่าตัดเข่าแล้วและอยู่ในระยะฟื้นฟู
• ผู้ดูแลและครอบครัว

✨ ฟีเจอร์หลัก
• 📅 ตารางฟื้นฟูตามระยะ — เป้าหมายและข้อควรระวังตั้งแต่ก่อนผ่าตัด ถึง 3 เดือนหลังผ่า
• 🦵 คลังท่าออกกำลังกาย — Quad Sets, SLR, Heel Slides, Ankle Pumps พร้อมตัวจับเวลานับเซ็ตอัตโนมัติ
• 📝 บันทึกอาการรายวัน — ระดับปวด ช่วงงอเข่า น้ำหนัก อาการบวม
• 💊 จัดการยา — ตั้งเวลายา รับการเตือนอัตโนมัติ บันทึกการทาน
• ⚖️ คำนวณ BMI — เห็นแนวโน้มน้ำหนักและผลต่อข้อเข่า
• ⚠️ Red-flag alert — สัญญาณเตือนที่ควรพบแพทย์ทันที

🔒 ความเป็นส่วนตัว
• ข้อมูลทั้งหมดเก็บในเครื่องของคุณเท่านั้น
• ไม่ส่งข้อมูลออก ไม่ติดตามตำแหน่ง
• ปฏิบัติตาม PDPA

⚠️ ข้อจำกัด
แอปนี้ไม่ใช่อุปกรณ์การแพทย์ เป็นเพียงเครื่องมือเสริมการรักษา
ไม่สามารถใช้แทนคำแนะนำของแพทย์หรือนักกายภาพบำบัดได้
หากมีอาการรุนแรง โปรดปรึกษาแพทย์ทันที
```

## Screenshot Requirements

### Google Play (อย่างน้อย 2 รูป สูงสุด 8 รูป)
- ขนาด: 1080×1920 หรือ 1080×2400 (9:16 หรือ 9:20)
- รูปแบบ: PNG หรือ JPEG
- เน้นแสดง:
  1. หน้าหลัก + Timeline
  2. หน้าออกกำลังกาย (Quad Sets timer)
  3. หน้า Diary (pain slider)
  4. หน้ารายการยา (มี ✓)
  5. หน้าโปรไฟล์

### App Store (บังคับ iPhone 6.7" — อย่างน้อย 1)
- ขนาด: 1290×2796 (iPhone 15 Pro Max)
- เพิ่มเติม: 6.5" (1242×2688), 5.5" (1242×2208) ถ้าจะ support iPhone รุ่นเก่า
- เน้นเหมือน Play Store

### วิธีถ่าย screenshot
- ใช้ Expo Go บน device → screenshot ปกติ
- หรือ Android emulator + `adb shell screencap`
- หรือ iOS simulator → File → New Screenshot
